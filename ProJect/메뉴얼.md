# 메뉴얼

* 1. Data Load

  ```
  1. Data Load 버튼을 클릭해 원하는 csv파일을 불러옵니다.
  2. 불러온 데이터의 정보를 직접 보면서 결측치 값, Column을 수정, 삭제 해줍니다.
  3. 자신이 원하는 형태로 만든 데이터를 csv형태로 저장해줍니다.
  ```

* 2. Training

  ```
  1. 불러온 데이터를 학습시키는 화면입니다.
  2. 학습 비율과, 검증 비율, 시험 비율을 원하는 숫자 만큼 입력하여 데이터를 분할시킵니다.
  3. x축과 y축에 대한 column을 설정하고 파라미터에 숫자로 입력하여 파라미터 학습실행을 시킵니다.
  4. 결과창에 RMSE를 통해 학습이 잘 되어있는지 확인하고, 옆에 그래프를 통해 학습이 잘 되어있는지를 볼 수 있습니다.
     (RMSE가 낮을 수록 학습이 잘 되어있습니다.)
  5. training버튼을 클릭하여 시험비율에 대한 예측이 잘 되어있는지를 볼 수 있습니다.
  6. + 버튼을 클릭하여 예측이 되어있는 부분을 자세히 볼 수 있습니다.
  7. 학습이 잘 되어있다면, 모델을 저장합니다.
  ```

* 3. Predict

  ```
  1. 예측하고 싶은 데이터와 모델을 불러옵니다.
  2. 불러온 데이터와 모델은 버튼 아래에 이름으로 띄어지고, 아래의 결과창에 불러온 데이터를 볼 수 있습니다.
  3. 실행 버튼을 클릭하여 예측을 실행합니다.
  4. 예측이 잘 되어있는지 확인할 수 있고, +버튼을 클릭하여 자세히 볼 수 있습니다.
  ```

  

* EXE 파일로 변환하기

```
pyinstaller Main.py

* console 창이 뜨지 않게 하려면 
	pyinstaller -w Main.py
	
* 실행파일 하나만 생성되게 하려면
	pyinstaller -F Main.py
를 해줄 수 있습니다.

혹시 Error가 나게 되는 경우 console 창을 생성하여 exe 파일을 실행시킨 상태에서 오류를 보며 해결할 수 있습니다.
```
